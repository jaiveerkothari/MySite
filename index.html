BuzzKill

Pebble App

This is an app for the Pebble smart watch that I developed at HackIllinois with my team. BuzzKill detects what website you are visiting on your laptop, checks whether it is a time-wasting website that you blacklisted (e.g. Facebook, Twitter, Reddit, etc.), and if it is, starts a timer. The Pebble vibrates every 15 seconds to remind you that you are still on that time wasting site. When you leave the time wasting site the timer and incessant buzzing stop. The app was coded in C using the online Pebble SDK. The app interfaces with the user’s laptop through a Chrome extension. The Chrome extension is coded in Javascript. Firebase was used as the database.

Top 100 Predictor

Machine Learning

This is a machine learning project that I did with my team at school. We pulled around 1000 songs from 2014 using Spotify’s online database. Then we tried to predict if a song would be in the Top 100 songs for that year. We used Weka for the machine learning algorithms. We used the j48 decision tree as it gave us the best f1 measure and highest accuracy of correctly classified instances.
